# Remote-Controller-IR
Remote Controller using IR receiver and IR Emitter

### Hardware used:
- Arduino Nano V2.0
- Logic Analyzer 24MHz 8CH
- KY-022
- IR Emitter (IR LED)

### IDE used:
- PlateformIO
